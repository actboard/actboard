▶ Trigger a Test Run
ActBoard collects results when your reporter publishes them. Runs are triggered by your CI or locally.
Local run
ACTBOARD_API_KEY=your_key npx playwright test
GitHub Actions
- name: Run Playwright Tests
  env:
    ACTBOARD_API_KEY: ${{ secrets.ACTBOARD_API_KEY }}
  run: npx playwright test
